## Option 2: GitHub Pages (If you use GitHub)

### Steps:

1. **Create a GitHub Account**
   - Go to: https://github.com
   - Sign up for a free account

2. **Create a New Repository**
   - Click "New repository"
   - Name it: `furqan-birthday` (or any name)
   - Make it **Public**
   - Click "Create repository"

3. **Upload Your Files**
   - Click "uploading an existing file"
   - Drag and drop all your files (index.html, style.css, script.js, assets folder)
   - Click "Commit changes"

4. **Enable GitHub Pages**
   - Go to repository Settings
   - Scroll to "Pages" section
   - Under "Source", select "main" branch
   - Click "Save"
   - Your site will be at: `https://yourusername.github.io/furqan-birthday`

## Option 3: Vercel (Fast & Easy)

### Steps:

1. **Go to Vercel**
   - Visit: https://vercel.com
   - Sign up with GitHub/Google/Email

2. **Deploy**
   - Click "Add New Project"
   - Import your GitHub repository OR
   - Drag and drop your folder
   - Click "Deploy"
   - Your site will be live in seconds!

---

## Option 4: Traditional Web Hosting

If you have web hosting (like cPanel, Bluehost, etc.):

1. **Upload Files via FTP or File Manager**
   - Upload all files to the `public_html` or `www` folder
   - Make sure `index.html` is in the root directory
   - Upload the `assets` folder with all images and videos

2. **Access Your Site**
   - Your site will be at: `https://yourdomain.com`
